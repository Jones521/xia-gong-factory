# 🚀 零命令行部署方案（推荐小白）

## 方案 1：Netlify 拖拽部署（最简单，5 分钟）

### 步骤

**1. 准备文件**

确保 `D:\openclaw-sim\website` 文件夹里有：
- ✅ index.html
- ✅ （可选）qr-code.png - 飞书群二维码

**2. 访问 Netlify Drop**

打开浏览器，访问：https://app.netlify.com/drop

**3. 拖拽文件夹**

把整个 `website` 文件夹拖到网页上的虚线框里

**4. 等待上传**

上传完成后，你会得到一个网址：
```
https://随机名字.netlify.app
```

**5. 自定义网址（可选）**

- 点击 "Site settings" → "Change site name"
- 输入你想要的名字，如 `xia-gong-factory`
- 网址变成：`https://xia-gong-factory.netlify.app`

## 方案 2：Vercel Git 部署（需 GitHub）

### 步骤

**1. 准备 GitHub 账号**

- 访问 https://github.com 注册账号

**2. 创建仓库**

- 点击右上角 "+" → "New repository"
- 仓库名：`xia-gong-factory`
- 勾选 "Public"
- 点击 "Create repository"

**3. 上传文件**

- 点击 "uploading an existing file"
- 把 `website` 文件夹里的文件拖进去
- 点击 "Commit changes"

**4. Vercel 连接**

- 访问 https://vercel.com/new
- 用 GitHub 登录
- 点击 "Import Git Repository"
- 选择 `xia-gong-factory` 仓库
- 点击 "Deploy"

**5. 等待部署**

1-2 分钟后，部署完成，获得网址：
```
https://xia-gong-factory.vercel.app
```

## 方案 3：Cloudflare Pages（免费，国内访问快）

### 步骤

**1. 注册 Cloudflare**

访问：https://dash.cloudflare.com/sign-up

**2. 创建 Pages 项目**

- 左侧菜单 "Workers & Pages" → "Create application"
- 选择 "Pages" → "Connect to Git"
- 选择 GitHub 仓库

**3. 配置构建**

- Build command: 留空（静态网站不需要）
- Build output: 留空
- 点击 "Save and Deploy"

## 部署后要做的事

1. **测试网站** - 打开网址，检查是否正常显示
2. **手机预览** - 用手机打开，检查响应式效果
3. **替换二维码** - 把飞书群二维码图片替换成你的
4. **复制网址** - 保存网址，用于小红书主页

## 遇到问题？

### Q: 上传失败
```
检查：
- 网络是否正常
- 文件是否完整
- 换个浏览器试试
```

### Q: 网站显示不正常
```
检查：
- index.html 是否在根目录
- 是否有 CSS/JS 引用路径错误
```

### Q: 国内访问慢
```
方案：
1. 使用 Cloudflare Pages
2. 或使用国内云服务（需备案）
```
