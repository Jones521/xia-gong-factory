## 云存储选择

| 平台 | 优势 | 劣势 | 适用 |
|------|------|------|------|
| **蓝奏云** | 不限速、免登录下载 | 单文件≤100MB | ✅ 安装包 |
| **阿里云盘** | 容量大、速度快 | 需登录 | 完整包 |
| **百度网盘** | 用户多 | 限速 | 备选 |
| **腾讯微云** | QQ 微信方便 | 限速 | 备选 |
| **七牛云** | 自定义域名 | 付费 | 专业 |

## 具体步骤

### 1. 上传文件到蓝奏云

```
1. 注册蓝奏云 (https://www.lanzou.com/)
2. 上传 install.ps1 和 install.sh
3. 创建"虾工厂"文件夹
4. 获取分享链接
```

### 2. 上传完整包到阿里云盘

```
1. 打包整个 skills/masterclaw 目录
2. 上传到阿里云盘
3. 创建分享 (设置永久有效)
4. 获取分享链接
```

### 3. 创建飞书云文档

```
1. 飞书 → 云文档 → 新建
2. 复制"飞书文档模板.md"内容
3. 填入下载链接
4. 设置"任何人可读"
5. 分享链接到群里
```

### 4. 飞书机器人自动回复

配置机器人关键词回复：

```
关键词：虾工厂
回复：🦐 欢迎了解虾工厂！
      使用教程：[文档链接]
      下载包：[蓝奏云链接]
      进群：[群链接]
```

## 安装脚本修改

脚本需要从云存储下载，而不是 GitHub：

```powershell
# install.ps1 - 蓝奏云版本
$DownloadUrl = "https://xxx.lanzou.com/xxx/masterclaw.zip"
$ZipPath = "$env:TEMP\masterclaw.zip"

Invoke-WebRequest -Uri $DownloadUrl -OutFile $ZipPath
# 后续解压逻辑不变
```
