## 安装步骤

### 方式 1：一键安装（推荐）

**Windows 用户：**

1. 下载 [install.ps1](下载链接)
2. 右键 → 使用 PowerShell 运行
3. 等待安装完成
4. 重启 OpenClaw

**Mac/Linux 用户：**

```bash
curl -sL 下载链接/install.sh | bash
```

### 方式 2：手动安装

1. 下载 [masterclaw.zip](下载链接)
2. 解压到 `C:\Users\你的用户名\.openclaw\skills\`
3. 重启 OpenClaw

---

## 使用

安装完成后，在 OpenClaw 中输入：

```
我想训一只虾
```

或者更具体：

```
帮我训一只理财虾，分析 A 股
帮我训一只客服虾，自动回复淘宝咨询
```
