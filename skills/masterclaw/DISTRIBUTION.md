```
│                      分发架构                                │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  Layer 1: GitHub 仓库 (代码托管)                            │

│  │ github.com/xia-gong-factory/masterclaw              │   │

```

## 实施步骤

### 阶段 1：GitHub 开源（建立信任）

```bash
# 创建仓库
github.com/xia-gong-factory/masterclaw

# 上传文件
- skills/masterclaw/*  (核心代码)
- README.md            (项目说明)
- LICENSE              (MIT 许可)
```

**目的：**
- 开源代码，建立信任
- 用户可查看代码再决定用不用
- Issue 收集需求

---

### 阶段 2：一键安装（降低门槛）

**Windows 用户：**
```powershell
iwr https://xia-gong-factory.github.io/install.ps1 -useb | iex
```

**Mac/Linux 用户：**
```bash
curl -sL https://xia-gong-factory.github.io/install.sh | bash
```

**目的：**
- 小白用户也能一键安装
- 无需懂 git/命令行

---

### 阶段 3：飞书群运营（转化付费）

**群内提供：**
- 使用教程
- 问题解答
- 案例分享
- 付费入口（¥9.9 体验价）

**转化路径：**
```
免费使用 → 遇到问题 → 群内解答 → 付费定制 → 长期用户
```
