## 这是什么？

Masterclaw 是 OpenClaw 的**训练师**，帮你把模糊的想法变成清晰的 AI 助手配置。

## 能训什么虾？

| 类型 | 功能 |
|------|------|
| 📊 数据虾 | 爬虫、Excel、报表 |
| 💰 理财虾 | 股票分析、基金跟踪 |
| 📝 文案虾 | 写文章、想选题 |
| 💬 客服虾 | 自动回复客户 |
| 🛍️ 电商虾 | 1688 爬取、Shopify 上架 |
| 🎨 设计虾 | P 图、抠图、滤镜 |
| 📈 运营虾 | 监控、告警、报告 |

## 快速开始

```bash
# 安装
clawhub install masterclaw

# 使用（对 OpenClaw 说）
"我想训一只理财虾"
```
