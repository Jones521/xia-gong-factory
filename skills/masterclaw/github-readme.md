## 🚀 快速开始

### 方式 1：一键安装脚本（推荐）

**Windows 用户：**
```powershell
iwr https://raw.githubusercontent.com/xia-gong-factory/masterclaw/main/install.ps1 -useb | iex
```

**Mac/Linux 用户：**
```bash
curl -sL https://raw.githubusercontent.com/xia-gong-factory/masterclaw/main/install.sh | bash
```

### 方式 2：手动安装

```bash
# 1. 克隆仓库
git clone https://github.com/xia-gong-factory/masterclaw.git

# 2. 复制到 openclaw skills 目录
cp -r masterclaw/skills/masterclaw ~/.openclaw/skills/

# 3. 重启 OpenClaw
openclaw restart
```
