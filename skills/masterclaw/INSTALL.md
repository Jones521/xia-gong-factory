# 虾工厂 Masterclaw - 安装说明

## 方式 1：Clawhub 安装（推荐）

```bash
clawhub install masterclaw
```

## 方式 2：手动安装

```bash
# 1. 克隆或下载本仓库
git clone https://github.com/xia-gong-factory/masterclaw.git

# 2. 复制到 openclaw skills 目录
cp -r masterclaw ~/.openclaw/skills/

# 3. 重启 openclaw
openclaw restart
```

## 方式 3：一键脚本

```bash
# Windows PowerShell
iwr https://xia-gong-factory.github.io/install.ps1 -useb | iex

# Linux/Mac
curl -sL https://xia-gong-factory.github.io/install.sh | bash
```
